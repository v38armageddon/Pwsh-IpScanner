## Usage
``Scan-IpAddress -IpAddress (-Port <Port>)`` : Scan a IP address.

``Scan-IpRange -StartIp <IP> -EndIp <IP> (-StartPort <Port>) (-EndPort <Port>)`` : Scan a range of IP addresses.
